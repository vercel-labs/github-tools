## Install the SDK

:::code-group
```bash [pnpm]
pnpm add @github-tools/sdk
```
```bash [npm]
npm install @github-tools/sdk
```
```bash [yarn]
yarn add @github-tools/sdk
```
```bash [bun]
bun add @github-tools/sdk
```
:::

## Add peer dependencies

### Required: AI SDK and Zod

The SDK relies on [AI SDK](https://ai-sdk.dev/docs) for tool execution and [Zod](https://zod.dev) for schema validation:

:::code-group
```bash [pnpm]
pnpm add ai zod
```
```bash [npm]
npm install ai zod
```
```bash [yarn]
yarn add ai zod
```
```bash [bun]
bun add ai zod
```
:::

### Optional: durable workflows

For **durable** agents with the [Vercel Workflow SDK](https://vercel.com/docs/workflow), add `workflow` and `@workflow/ai` when you import from `@github-tools/sdk/workflow` and use `createDurableGithubAgent`:

:::code-group
```bash [pnpm]
pnpm add workflow @workflow/ai
```
```bash [npm]
npm install workflow @workflow/ai
```
```bash [yarn]
yarn add workflow @workflow/ai
```
```bash [bun]
bun add workflow @workflow/ai
```
:::

See [Durable workflows (Vercel Workflow)](/guide/durable-workflows) for the full pattern.

## Set your GitHub token

The SDK reads `GITHUB_TOKEN` from your environment automatically. Create a `.env` file at the root of your project:

```bash [.env]
GITHUB_TOKEN=github_pat_xxxxxxxxxxxx
```

You can also pass the token explicitly if you prefer:

```ts [explicit-token.ts]
createGithubTools({ token: 'github_pat_xxxxxxxxxxxx' })
```

::warning
Never commit tokens. Use `.env` locally and secret managers (Vercel, GitHub Actions secrets) in CI/production.
::

## Choose a token type

| Token type | When to use | Scope control |
|---|---|---|
| [Fine-grained PAT](https://github.com/settings/personal-access-tokens/new) | Production assistants | per-repository, per-permission |
| Classic PAT | Quick prototyping | broad, org-level |

See [Configure token scopes](/guide/token-permissions) for a detailed permission matrix.

## Verify the setup

Run a minimal script to confirm the SDK initializes and a read tool resolves:

```ts [verify.ts]
import { createGithubTools } from '@github-tools/sdk'

const tools = createGithubTools({ preset: 'repo-explorer' })

console.log('Tools loaded:', Object.keys(tools).join(', '))
```

If the script prints tool names, your environment is ready. Continue with [Run your first call](/guide/quick-start).
